# EarnedReach Journey Board - Design System

## Design Philosophy: **Swiss Modernism meets Digital Clarity**

This design draws from **Swiss International Style** principles—precision, hierarchy, and functional beauty—adapted for an interactive digital experience.

### Core Principles
1. **Grid-Based Precision**: All elements align to an 8px grid system
2. **Typographic Hierarchy**: Clear distinction between headings, labels, and body text
3. **Functional Color**: Color serves meaning, not decoration
4. **Generous Whitespace**: Breathing room enhances comprehension
5. **Subtle Motion**: Transitions guide attention without distraction

### Color Philosophy
- **Primary (Blue)**: Trust, professionalism, progress (`#2563EB`)
- **Accent (Amber)**: Highlights, achievements, milestones (`#F59E0B`)
- **Neutrals**: Clean canvas with depth (`#F8FAFC` to `#1E293B`)
- **Semantic**: Success (green), Warning (yellow), Info (blue)

### Typography System
- **Display**: DM Sans Bold (24-32px) - Section headers
- **Heading**: DM Sans SemiBold (18-20px) - Card titles
- **Body**: Inter Regular (14-16px) - Content
- **Label**: Inter Medium (12-14px) - Metadata

### Layout Paradigm
**Horizontal Timeline Flow** - The journey progresses left-to-right, mimicking natural reading patterns and emphasizing forward momentum.

### Signature Elements
1. **Connection Lines**: Animated SVG paths showing flow between stages
2. **Stage Cards**: Elevated cards with subtle shadows and hover states
3. **Progress Indicators**: Circular badges showing completion status
4. **Interactive Tooltips**: Contextual information on hover

### Animation Guidelines
- **Entrance**: Staggered fade-in with subtle upward motion (0.3s ease-out)
- **Hover**: Gentle lift with shadow expansion (0.2s ease)
- **Transitions**: Smooth color and size changes (0.15s ease-in-out)
- **Connections**: Animated stroke-dasharray for path drawing effect

This design creates a professional, trustworthy experience that reflects EarnedReach's strategic positioning while remaining accessible and easy to navigate.
